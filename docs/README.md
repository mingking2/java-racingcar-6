# 미션 - 자동차 경주

## 요구 사항

1. 게임시작
2. 게임준비
    - 자동차 이름 입력을 위한 안내 메세지를 출력한다.
    - 자동차 이름 입력받는다.
      - 공백인가?
      - 길이가 5이하인가?
      - 중복된 값이 있는가?
    - 시도 횟수 입력을 위한 안내 메세지를 출력한다.
    - 시도 횟수 입력받는다.
3. 게임진행
    - 무작위 값을 뽑는다.
    - 값에 따라 전진 여부를 결정한다.
    - 전진한다.
    - 전진한 결과를 출력한다.
4. 게임 우승자 발표
   - 전진한 값을 비교한다.
   - 가장 많이 전진한 자동차를 출력한다.
